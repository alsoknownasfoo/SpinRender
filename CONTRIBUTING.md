# Contributing to SpinRender

Contributions are welcome! Whether you're fixing a bug, adding a new camera preset, or improving the 3D preview, your help is appreciated.

## How to Contribute

1.  **Fork the repository** on GitHub.
2.  **Clone your fork** locally:
    ```bash
    git clone https://github.com/yourusername/SpinRender.git
    ```
3.  **Create a branch** for your feature or fix:
    ```bash
    git checkout -b feature/your-awesome-feature
    ```
4.  **Install development dependencies**:
    ```bash
    pip install -r requirements_dev.txt  # If applicable
    ```
5.  **Make your changes** and ensure everything works correctly.
6.  **Run tests** (if any exist):
    ```bash
    pytest
    ```
7.  **Commit your changes** with a clear and descriptive commit message.
8.  **Push to your fork** and **submit a Pull Request**.

## Contribution Ideas

- **Camera Presets**: Add more creative camera loop presets.
- **Lighting**: Improve the lighting model or add more environment presets.
- **Optimization**: Make the render process faster or more efficient.
- **UI/UX**: Refine the main panel for better usability.

## Code of Conduct

Please be respectful and helpful in all interactions within the project.
