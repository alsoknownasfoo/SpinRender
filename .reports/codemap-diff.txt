## Architecture Patterns (Current)

- **Singleton**: Theme, Locale, TextStyles, SpinLogger
- **MVC-ish**: Settings (Model) ↔ ParameterController (Controller) ↔ CustomControls (View)
- **Controller**: RenderController (new) separates rendering logic from UI
- **Registry**: ControlRegistry tracks controls for batch operations
- **Observer (implicit)**: Theme/Locale hot-reload via timer polling
- **Lazy Loading**: OpenGL/trimesh imports deferred until after dependency check
- **Hot-Reload**: Theme/Locale file watchers update UI in-place
- **Factory**: CustomButton, CustomSlider, etc. create themed controls
- **Custom Events**: wx.PyEventBinder for component communication
- **Hot-Reload**: Preset thumbnails cached with LRU eviction
